TideAegis 是一款综合性的、简单易懂的、包含当前中国主要流行毒株的PRRSV遗传演化分析工具，同时该软件也精心整合了PCV2、PCV3、GETV、CSFV、PRV、PHEV、PoRV以及PCMV等多个猪呼吸道主要病毒遗传演化分析的参考序列数据库。用户只需要根据自己的序列（查询序列）选择相应的数据库（参考序列），即可快速分析查询序列与参考序列的关系。同时，TideAegis整合了序列数据库(用户可以对数据库进行增删改查等操作)、病毒基因组变异分析、载体测序数据预处理(去除载体序列)、荧光PCR数据处理等多个功能。旨在为解决病原检测的“最后一公里”——数据分析。笔者坚信TideAegis 在不断迭代和升级后终将成为流行病学从业者的有力工具。(预分发版本， 数据库将在后期完善)

TideAegis is an intuitive, all-in-one tool for genetic and evolutionary analysis of PRRSV, tailored to include the major strains currently circulating in China. It also thoughtfully integrates curated reference sequence databases for other key swine respiratory viruses, including PCV2, PCV3, GETV, CSFV, PRV, PHEV, PoRV, and PCMV.
With TideAegis, users can simply select the appropriate reference database and load their query sequences to quickly assess genetic relationships. Beyond this core functionality, the platform also supports database management (including add/edit/delete/search operations), viral genome variation analysis, vector sequence removal from sequencing data, and fluorescence qPCR data processing.
TideAegis is designed to bridge the “last mile” of pathogen detection—data analysis. With ongoing iteration and refinement, it is poised to become a powerful tool for epidemiologists and researchers in the field.
